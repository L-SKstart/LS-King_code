# AI 协作工作空间模板

> 一套经过实战验证的 **AI 协作工作空间规范 + 目录标准 + 沟通体系**
> 让 AI 帮你干活，而不是给你添乱

---

## 这是什么

这是一套**可复用的工作空间模板**，包含：

| 组件 | 说明 |
|------|------|
| 📜 **22条规则体系** | AI 该做什么、不该做什么、违规怎么处理 |
| 📁 **标准目录骨架** | 文件该放哪、文档怎么分类 |
| 🤝 **协作格式规范** | 人和AI、AI和AI之间怎么高效沟通 |
| 📋 **模板文件** | 拿来改几个占位符就能用 |

## 适合谁

- **个人开发者** — 想用 AI 辅助编码，但不知道怎么系统化管理
- **小团队** — 多个人+多个AI 协作，需要统一规范
- **非技术人员** — 想用 AI 帮自己干活，但不知道从哪开始

## 怎么用

```bash
# 1. 下载或克隆此模板
# 2. 打开 docs/说明/快速上手指南.md 按步骤操作
# 3. 批量替换 [占位符] 为你自己的信息
# 4. 复制到你的工作空间，启动 AI
```

## 核心理念

> **不是 AI 不够强，是没给 AI 定规矩。**
>
> 这套模板的核心就是：把 AI 当成团队成员来管——有规范、有记录、有追责。
