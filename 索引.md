## 🔥 快速搜报错（搜关键词直达解法）

### MySQL（4条）→ `错误分类记录.md` #1~#4

| 你看到的关键词 | 编号 | 一句话解法 |
|------|:--:|------|
| `Access denied` / 密码错误 | #1 | 改 MySQL 密码 + bootstrap.yml 同步 |
| `Host is blocked` / flush-hosts | #2 | `FLUSH HOSTS` + 调大 max_connect_errors |
| `Public Key Retrieval` | #3 | JDBC URL 加 `allowPublicKeyRetrieval=true` |
| `Data too long for column` | #4 | `ALTER TABLE ... MODIFY COLUMN ... mediumtext` |

### Shell / OSS 脚本（18条）→ #1~#18 + #24~#25

| 你看到的关键词 | 编号 | 一句话解法 |
|------|:--:|------|
| 日期被污染 / 只处理当天 | #2 | `log()` 改 `>&2` 输出到 stderr |
| `oss_ls` 太慢 / 全量列出 | #3 | 加 `--include` 服务端过滤 |
| `dat: command not found` | #4 | `date` 误写为 `dat`，同步新版脚本 |
| 循环只跑一次就退出 | #5 | `((succ++))` → `((succ++)) \|\| true` |
| "无对应 output" / output 找不到 | #6 | 动态匹配后缀 `grep -E ".*_output.*\.tar\.gz"` |
| `tar -xOf` 找不到文件 | #7 | 拼内层目录 `${fname%.tar.gz}/BasicInfo/...` |
| oss_download 下载失败 | #8 | `--include` 用时间戳前缀 + `output_name` 去全路径 |
| `awk index()` 误匹配 | #9 | `$NF ~ "/日期"` 替代 `index($NF, "日期")` |
| OSS 桶不对 | #10 | 切凭据和路径为 `ydxt-2` |
| `all_remote` 过滤后为空 | #11 | `include_arg` 变量不加引号导致 `*` 被 glob |
| `grep '\.tar\.gz$'` 没结果 | #12 | 去 `$` 锚点，兼容 CRLF 行尾 |
| "成功:1 跳过:0" 计数不对 | #13 | `return 0` → `return 1`（跳过不应计成功） |
| oss_download 输出了报错 | #14 | `>/dev/null` → `>/dev/null 2>&1` |
| 误删根目录文件 | #15 | `rm -rf "${VAR:?}"` 加 `:?` 保护 |
| for 循环 mv 失败后脚本退出 | #16 | 每个 `mv` 后加 `\|\| true` |
| output 文件检测失效 | #17 | `output_name` 的 grep 也去 `$` 锚点 |
| tar 解压失败脚本静默崩溃 | #18 | `if ! tar ...; then log "ERROR" ...; return 1; fi` |
| output 解压后目录全倒入 DHM_OUT | #24 | glob 拼了两次 `_output`，去重 |
| `rm -rf ${in_target}` 缺防护 | #25 | 改为 `${in_target:?}` 和 `${out_target:?}` |

### Java / X11（5条）→ #19~#23

| 你看到的关键词 | 编号 | 一句话解法 |
|------|:--:|------|
| `no libsigar-amd64-linux.so` | #19 | `-Djava.library.path=/opt/glzz/` |
| `FileNotFoundException: ClientConfig` | #20 | 先 `cd /opt/glzz` 再启动 |
| `Can't connect to X11` / `No protocol` | #21 | `xhost +SI:localuser:gluser` |
| `HeadlessException` / `No X11 DISPLAY` | #22 | `export DISPLAY && java` 同一行 |
| `sudo -u gluser` 报错 | #23 | 去掉 `sudo -u`，gluser 自己跑 |
