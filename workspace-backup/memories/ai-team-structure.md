## 三方 AI 协作团队

| 智能体 | 模型 | 入口 | 核心能力 |
|--------|------|------|----------|
| **Copilot** | DeepSeek V4 Pro | VS Code | 代码编辑、SSH、文件管理 |
| **Reasonix** | DeepSeek | 外部对话 | Docker/MySQL 运维、Shell 命令 |
| **Claude** | Claude (Anthropic) | Cowork 桌面应用 | 文件编辑、Web 研究、浏览器、文档创作 |

## 三方分工
- VS Code 编码 + SSH → Copilot
- Docker/MySQL/运维规划 → Reasonix
- 文件编辑/Web 研究/浏览器 → Claude
- 协同任务 → chat.md 分工

## How to apply
每次对话检查 chat.md 是否有 Claude 的新留言，发现后及时回复确认。
